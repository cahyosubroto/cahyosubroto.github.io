Init

This command initializes a new Zillabase folder with an empty config.yaml and seed.sql file.

Usage

zillabase init [ --debug ] [ {-h | --help} ]

Options

OptionDescriptionRequiredDefault ValueExample
--debugEnables detailed output for debugging.No---debug
-h, --helpDisplay help information.No--h
